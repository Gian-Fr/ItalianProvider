| Provider           | Type      | Search | View | Notes |
|--------------------|-----------|-------|----|-------|
| GuardaSerie        | Tv Series | ✅     | ✅ |

| SoraStreamItaliano | Tv Series / Movies | ✅     | ✅ | some sources may not work - Only movies work

| TantiFilm          | Tv Series / Movies | ✅     | ❌ | temporarily deactivated
